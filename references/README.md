# ClearLedger Advisory — Premium Excel Toolkit (v2)

13 standalone, sellable advisory products. Rebuilt to premium standard from the v1 drafts.

## What changed from v1 → v2
- **Numbers are real numbers** (v1 stored KPI values as text — couldn't compute, sort, or chart, and triggered Excel warnings)
- **Proper number formatting** — currency (€/$), percentages, decimals, with units in headers
- **Live RAG status** on every KPI scorecard — direction-aware formulas + conditional formatting (green On Target / amber Near / red Off Target)
- **Auto-rating Credit Scoring Model** — enter 1–5 scores, get a live A/B/C/D rating with data bars
- **Color-coded RACI** (R/A/C/I), numbered SOP step badges, stepped Instructions cards
- **Refined navy + gold branding** on every cover, consistent with the ClearLedger app
- **Landscape fit-to-width page setup** — prints and exports to PDF cleanly
- **Zero formula errors** across all 13 files (verified via recalculation)
- **Working calculators preserved** — Business Case Builder (NPV/IRR/payback, 140 formulas) and BPO/SSC/GBS Evaluation (cost models, 98 formulas) retain their full interconnected formula web
